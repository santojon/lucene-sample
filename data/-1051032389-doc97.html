Descrição do Jogo

Participe de uma épica batalha entre cavaleiros medievais neste incrível jogo de luta para 3 jogadores.
Super Heróis leva você para uma cidade invadida por bandidos.Seu objetivo é expulsá-los de lá para trazer a paz de volta a população.
Escolha o seu personagem favorito:Um guerreiro,um arqueiro ou um lutador.Cada um deles possui suas próprias características e habilidades especiais.
Quando terminar,em seguida,comece a batalha.Você pode usar socos,chutes e golpes especiais,além de armas e outros artefatos para derrotar os bandidos.
Veja abaixo as instruções para controlar o seu jogador.

Controles de Jogo

Jogador 1
Use as teclas W,R,A,C,D,F e X para jogar

Jogador 2
Use as teclas I,F,J,K,L para jogar.

Jogador 3
Use as teclas 1,5,3,2,0,+ e - para jogar
